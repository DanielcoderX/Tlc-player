# TLC-player
Lightweight, Modern and Fast CLI Music Player with VLC core, written in Python !

# Running
1. Installing dependencies :
```bash
pip3 install -r requirements.txt
```
2. Run TLC with :
```bash
python3 tlc.py <music-path>
```
